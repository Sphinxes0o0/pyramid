## Metadata
- **Author:** 邓俊辉（清华大学）
- **Edition:** 第3版
- **Publisher:** 清华大学出版社

## Core Content
清华大学数据结构经典教材（C++版，669页）。

**主要内容**：
- 数据结构基础：向量、链表（列表、栈、队列）
- 树结构：二叉树、AVL树、红黑树、B树
- 图结构：图的遍历、最短路径、最小生成树
- 排序算法：内部排序、外部排序
- 查找：二叉搜索树、平衡二叉树、哈希表
- 优先队列：堆（完全二叉堆、左式堆、二项堆）
- 字符串算法：KMP、Trie、后缀数组
- 高级数据结构：跳表、伸展树、AA树

所有实现使用 C++，体现现代 C++ 风格的数据结构设计。
