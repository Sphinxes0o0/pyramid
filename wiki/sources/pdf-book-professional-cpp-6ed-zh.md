# Professional C++ 第六版（中文）

## 核心内容

- **C++20/23 新特性**：模块、Ranges、Concepts、Coroutines、std::format
- **工程实践**：代码组织、命名规范、性能优化、编译器 flags
- **泛型编程**：模板技法、模板与继承、CRTP、type erasure
- **并发**：std::async、future/promise、atomic、memory_order、死锁避免
- **设计模式**：GoF 模式在 C++ 中的实现（单例/工厂/策略/观察者）
- **STL 高级用法**：自定义分配器、迭代器适配器、容器内部机制
- **调试与工具**：Valgrind、Sanitizers、profiler
