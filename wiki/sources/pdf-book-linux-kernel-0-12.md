# Linux内核0.12完全注释

## 核心内容

赵炯著作，对 Linux 0.12 内核全部源代码进行详细注释：

- **PC 硬件基础**：运行 Linux 系统的 PC 硬件组成结构
- **汇编基础**：80x86 汇编语言与 GCC C 扩展
- **内核源码注释**：内存管理、进程调度、文件系统、VFS 等
- 内核发展历史与各版本主要区别

## 关键要点

- 选择 0.12 版本作为研究对象原因：代码量适中，能正常编译运行，包含 Linux 工作原理精髓
- 适合作为 Linux 内核入门读物，理解基本工作机理后再研究现代内核
- 作者网站 www.oldlinux.org 提供大量 Linux 历史资料
