# Modern C++ Tutorial 中文版

## 核心内容

- **语言特性**：auto、lambda、委托构造、移动语义、右值引用、折叠表达式
- **类型系统**：variant、optional、any、tuple、string_view
- **智能指针**：unique_ptr/shared_ptr/weak_ptr / make_unique/make_shared
- **并发**：atomic、memory_order、future/promise、async、thread
- **Ranges/Views**：管道式算法、惰性求值
- **Concepts**：模板约束、requires 子句
- **Coroutines**：co_await/co_yield/co_return 基础
- **标准库**：chrono、filesystem、random、regex 常用场景
