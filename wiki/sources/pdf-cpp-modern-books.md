# Modern C++ Programming Books

## 1. The Book of Modern C++ (2nd Edition)

**作者**: lkimuk404, 何荣华, 邹启翔, 水滴会飞 | **出版**: 2024 | **页数**: 1053

社区协作编写的 C++20/23 高级教程，完全免费开源。涵盖 C++20/23 新特性、高级主题和多位专家的实践经验。

### 核心主题
- C++20/23 新特性深度讲解
- 现代 C++ 编程实践
- 多专家视角的高级主题

## 2. Professional C++ 6th Edition (中文翻译)

**作者**: Marc Gregoire | **译者**: 陈晓伟 | **页数**: 1247

C++ 领域最全面的专业教程之一，覆盖从基础到高级的完整知识体系。第6版更新至 C++20 标准。

### 核心主题
- C++ 基础和标准库
- C++20 新特性（Concepts、Modules、Coroutines、Ranges）
- 面向对象设计与模式
- 软件工程实践与调试

## 3. 21st Century C++

**作者**: Bjarne Stroustrup (C++ 之父) | **出版**: 2022 | **页数**: 26

CACM 论文，阐述现代 C++ 的核心设计理念：资源管理、生命周期管理、错误处理、模块化和泛型编程。强调类型安全和性能的统一。

### 核心主题
- 资源管理与 RAII
- 生命周期安全
- 错误处理模式
- 模块化设计
- 泛型编程与 Concepts
- C++ Core Guidelines

## 4. Modern C

**作者**: Jens Gustedt (INRIA) | **出版**: 2018 | **页数**: 320

现代 C 语言权威教程，阐述 C11/C17 标准下的现代 C 编程实践。虽为 C 语言书籍，但与 C++ 紧密相关。

### 核心主题
- C11/C17 新特性
- 内存模型与并发
- 类型泛型编程
- 安全编程实践

## 5. C++17 — The Complete Guide (中文翻译)

**作者**: Nicolai M. Josuttis | **译者**: 陈晓伟 | **页数**: 400

C++17 标准特性完全指南。结构化绑定、if/switch 初始化、折叠表达式、if constexpr、std::optional/variant/any、文件系统库、并行算法等。

### 核心主题
- 结构化绑定
- if/switch 带初始化
- 折叠表达式
- if constexpr
- std::optional/variant/any
- 文件系统库
- 并行算法（执行策略）

## 6. Large-Scale C++ Software Design

**作者**: John Lakos | **出版**: 1996 | **页数**: 870

C++ 大规模软件设计经典著作，聚焦物理设计（physical design）、组件（components）、层级（hierarchies）和依赖管理。

### 核心主题
- 物理设计与逻辑设计分离
- 组件层次结构
- 依赖分析与管理
- 循环依赖消除
- 头文件组织与预编译
- 大规模重构策略
