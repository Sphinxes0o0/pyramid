# C Programming Language Books

## 1. The C Programming Language, 2nd Edition

**作者**: Brian W. Kernighan, Dennis M. Ritchie | **页数**: 288 | **出版**: 1988

C 语言圣经，由语言创造者亲笔撰写。第2版覆盖 ANSI C (C89/C90) 标准。本书 PDF 为扫描版，内容为图像格式。

### 核心主题
- 基本概念：类型、运算符、表达式
- 控制流
- 函数与程序结构
- 指针与数组
- 结构体与联合体
- 输入输出
- UNIX 系统接口

## 2. C in a Nutshell, 2nd Edition

**作者**: Peter Prinz, Tony Crawford | **出版**: 2015 | **页数**: 823

O'Reilly 出品的 C 语言完整参考手册，覆盖 C11 标准。包含语言参考、标准库参考和 C 运行时环境三大部分。

### 核心主题
- C 语言基础与语法
- 类型系统：基本类型、枚举、结构体、联合体
- 指针与数组深度分析
- 函数与链接
- 预处理指令
- C 标准库详解（<stdio.h>, <stdlib.h>, <string.h>, <math.h> 等）
- C11 新特性：泛型表达式、多线程支持（<threads.h>）、原子操作（<stdatomic.h>）
- 运行时环境与 ABI
