# Unix Programming Tools

## 核心内容

Stanford 课程文档，介绍 Unix 编程工具链的入门知识：

- **编辑-编译-链接-调试** 周期（edit-compile-link-debug cycle）
- **gcc** — C/C++ 编译器
- **make** — 构建自动化
- **gdb** — 调试器
- **emacs** — 编辑器
- **Unix shell** — 命令行环境

## 关键要点

- 适合初学者了解 Unix 开发环境的基本组成
- 内容较为基础，重点在工具之间的协作关系
