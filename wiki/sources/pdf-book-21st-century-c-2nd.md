# 21st Century C, 2nd Edition

## 基本信息
- **作者**: Ben Klemens
- **出版**: O'Reilly, 2nd Edition, 2018
- **页数**: ~408
- **语言**: English
- **主题**: 现代 C 编程实践（Autotools、GDB、第三方库集成）

## 核心内容
- C 语言的现代编程范式（相比 K&R C 的演进）

- POSIX/GNU 扩展
- 动态库与静态库
- C 与 Python/Lua 等脚本语言交互
