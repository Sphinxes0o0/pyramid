# 设计模式笔记

## Overview

系统化的设计模式学习笔记，涵盖 SOLID 原则、创建型模式（5种）、结构型模式（7种）、行为型模式（11种）共 23 种设计模式。课程含大量实战场景解析，探讨如何通过设计原则减少代码间相互影响。

## Key Topics

- **SOLID 原则**：单一职责、开闭原则、里氏替换、接口隔离、依赖倒置
- **创建型模式**：单例、建造者、抽象工厂、工厂方法、原型模式
- **结构型模式**：适配器、桥接、组合、装饰器、外观、享元、代理模式
- **行为型模式**：访问者、模板方法、策略、状态、观察者、备忘录、中介者、迭代器、解释器、命令、责任链模式
- **设计原则**：反转原则、惯例原则、分离原则、契约原则
- **实战场景**：API 网关高可用、营销推荐、有限状态机、聊天历史记录等
