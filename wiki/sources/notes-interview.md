# 面试准备笔记

## Overview

整理自 jwasham/coding-interview-university 的系统性面试准备指南，涵盖方法论、问题解决模式、系统设计基础、NP完全性、位操作技巧、进阶数据结构等主题。目标公司包括 Amazon、Facebook、Google、Microsoft 等大型科技公司。

## Key Topics

- **方法论**：面试准备策略、学习路线图、软素质培养
- **问题解决模式**：编码面试通用解题思路（滑动窗口、双指针、DFS/BFS等）
- **系统设计基础**：可扩展性设计、数据处理、系统架构
- **NP完全性与近似算法**：复杂度理论、近似解法
- **位操作技巧**：位运算技巧与实战应用
- **进阶数据结构**：Trie、Skip Lists、平衡搜索树
- **推荐资源**：书籍、刷题平台、Mock 面试资源
