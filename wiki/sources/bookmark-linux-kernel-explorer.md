# Linux Kernel Explorer

来源: [reverser.dev/linux-kernel-explorer](https://reverser.dev/linux-kernel-explorer) — 在线内核源码浏览器

## 核心功能

| 功能 | 描述 |
|------|------|
| **符号搜索** | 函数/变量/结构体快速定位 |
| **类型查找** | struct/enum/typedef 查找 |
| **交叉引用** | XREF 导航，理解调用关系 |
| **内核版本** | 支持多内核版本切换 |

## 资源特点

- **在线工具**: 无需本地配置，直接浏览器使用
- **快速定位**: 适合日常内核开发/学习时快速查找
- **轻量替代**: 可替代笨重的sourcegraph for kernel

## 适用场景

- 阅读内核代码时快速跳转
- 查找某函数在内核中的实现位置
- 理解内核符号和类型的交叉引用
