# Data Structures and Algorithms in C++, 2nd Edition

## 核心内容

- **复杂度分析**：大O/Ω/Θ、递归方程求解、主定理
- **数据结构**：数组/链表/栈队列/树（二叉/B树/AVL/红黑）/堆/哈希表
- **图算法**：BFS/DFS/Prim/Kruskal/Dijkstra/Floyd
- **排序**：插入/希尔/堆/归并/快速/桶/基数排序，含复杂度分析
- **动态规划**：LCS/矩阵链乘/0-1背包/最长递增子序列
- **算法设计**：分治、贪心、回溯、摊销分析
- **C++ 实现**：每种数据结构都有完整可运行的 C++ 代码
