# Foundations of Visual Computing

## Core Content

Online book treating visual computing as "signal transductions across different domains — optical, analog, digital, and semantic."

### Key Topics
- **Human Vision**: Visual perception, color spaces, contrast sensitivity
- **Rendering**: Ray tracing, rasterization, physically-based rendering
- **Imaging**: Image formation, sensors, computational photography
- **Display Technologies**: LCD, OLED, HDR displays, color management

### Approach
Crosses disciplinary boundaries: optics, signal processing, computer graphics, perception psychology.
