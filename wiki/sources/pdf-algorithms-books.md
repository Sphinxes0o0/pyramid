## Part A: 算法与数据结构详细

### 1. Data Structures & Algorithms in C++, 2nd Ed
- **作者**: Michael T. Goodrich (UC Irvine), Roberto Tamassia (Brown), David Mount (UMD)
- **出版**: John Wiley & Sons, 2nd Edition
- **大小**: 17.8 MB
- **内容**: C++标准库实现的数据结构和算法全面覆盖
  - 章节: Mathematical Preliminaries, Array/Vector/Matrix, Stacks/Queues, Lists, Trees, Binary Search Trees, AVL Trees, Splay Trees, Red-Black Trees, Skip Lists, Scapegoat Trees, Treaps, Hash Tables, Priority Queues, Map/Dictionary, Sort Algorithms, Selection, Graph Traversal, MST, Shortest Path, Network Flow
- **特点**: 每个算法给出O(n)分析；C++实现直接可用
- **状态**: PDF文本完整

### 2. Data Structures & Algorithms in C++, 3rd Ed
- **作者**: Michael T. Goodrich (UC Irvine), Roberto Tamassia (Brown), David Mount (UMD)
- **出版**: John Wiley & Sons, 3rd Edition
- **大小**: 44.0 MB
- **内容**: DS&A C++版第3版，全面更新
- **状态**: PDF文本完整

### 3. Algorithms in C, 3rd Ed (Full)
- **作者**: Robert Sedgewick (Princeton)
- **出版**: Addison-Wesley, 3rd Ed
- **大小**: 30.4 MB
- **内容**: 算法全卷（含排序、搜索、字符串、几何等）
- **语言**: C语言实现
- **状态**: 扫描版

### 4. Algorithms in C — Part 5: Graph Algorithms
- **作者**: Robert Sedgewick (Princeton)
- **出版**: Addison-Wesley, 3rd Ed, 2002
- **大小**: 1.1 MB
- **内容**: 图算法专题卷
  - 涵盖: DFS/BFS、有向图、无向图、最小生成树(Prim/Kruskal)、最短路径(Dijkstra/Floyd/Bellman-Ford)、网络流(Ford-Fulkerson/Edmonds-Karp)
- **语言**: C语言实现
- **状态**: 扫描版，图算法全

### 5. The Art of Computer Programming, Vol.1: Fundamental Algorithms, 3rd Ed
- **作者**: Donald E. Knuth (Stanford)
- **出版**: Addison-Wesley, 3rd Edition
- **大小**: 6.7 MB
- **内容**: 计算机程序设计艺术第一卷
  - MMIX指令集、信息结构（线性表、链接表、循环表、二叉树）、二叉树遍历、递归、栈与子程序、随机器、算术运算
- **地位**: 计算机科学史上最经典算法著作
- **状态**: PDF文本完整

### 6. 我的第一本算法书
- **作者**: 石田保辉, 宫崎修一
- **出版**: 人民邮电出版社, 2018
- **大小**: 8.9 MB
- **内容**: 图解算法入门
  - 7个数据结构: 链表、数组、栈、队列、哈希表、二叉树、图
  - 26个基础算法: 冒泡排序、二分查找、广度优先搜索、哈希函数、迪菲-赫尔曼密钥交换、k-means算法
- **特点**: 大量图片、步骤图详解，直观易懂
- **状态**: PDF文本可读

## Part B: 编程面试详细

### 7. Cracking the Coding Interview, 6th Ed
- **作者**: Gayle Laakmann McDowell (CareerCup创始人/Google前工程师)
- **出版**: CareerCup, 2015
- **页数**: 708
- **内容**: 189道编程面试题及详解
  - 面试流程与策略（简历、行为问题、技术问题）
  - 数据结构专题（数组/字符串、链表、栈/队列、树/图）
  - 算法专题（递归、排序、搜索、DP）
  - 系统设计与可扩展性
  - 进阶主题（OOD、数据库、线程与锁）
  - 189道精选面试题
- **特点**: 每道题Hint + Solution + 复杂度分析；大厂面试真题来源
- **状态**: PDF文本完整

### 8. Elements of Programming Interviews (EPI) — Sampler
- **作者**: Adnan Aziz (UT Austin/Google), Tsung-Hsien Lee (Google), Amit Prakash (ThoughtSpot/Google/Microsoft)

- **页数**: 191 (sampler)
- **内容**: EPI面试指南概览

  - Part II: 领域问题 — 设计问题、语言问题、OOD、常用工具
  - Part III: 荣誉题 — 进阶问题
- **特点**: 结构化方法论（brute-force→optimized）；所有代码Python 3
- **状态**: PDF完整sampler

## 核心概念覆盖

### 数据结构
- 线性结构: 数组、链表、栈、队列
- 树形结构: 二叉树、BST、AVL、红黑树、堆、Treap、Skip List
- 图结构: 有向图、无向图、加权图
- 哈希: 哈希表、哈希函数、冲突处理
- 字符串: 串匹配、Trie

### 算法
- 排序: 冒泡、插入、归并、快排、堆排、桶排
- 搜索: 二分查找、DFS、BFS
- 图算法: 最短路径(Dijkstra/Floyd/Bellman-Ford)、MST(Prim/Kruskal)、网络流
- 递归与分治
- 动态规划
- 贪心算法

### 面试专题
- 问题解决模式: 双指针、滑动窗口、DFS/BFS回溯、DP状态机
- 系统设计: 可扩展性、负载均衡、CAP、一致性哈希
- OOD: SOLID原则、设计模式
- 复杂度分析
