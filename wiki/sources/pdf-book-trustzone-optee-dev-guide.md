# 手机安全和可信应用开发指南：TrustZone 与 OP-TEE 技术详解

## 核心内容

- **TrustZone 架构**：安全世界(Secure World)与正常世界(Normal World)隔离
- **TEE 与 Rich OS**：Linux/Android 与 TEE 的交互、Trusty/OP-TEE/s6
- **OP-TEE**：开源 TEE 实现、GP TEE API、Trusted Application 开发
- **安全启动**：ATF(BL1/BL2/BL31)、镜像验证、信任链
- **安全存储**：SFS/RPMB、密钥包装、安全文件系统
- **密码学**：Secure Enclave、硬件随机数、RNG 驱动
- **调试**：OP-TEE debug、secure monitor call、REE-TEE IPC
