# The Linux Programming Interface (TLPI)

## 基本信息
- **作者**: Michael Kerrisk (man7.org)
- **出版**: No Starch Press, 2010
- **页数**: ~1552
- **语言**: English
- **主题**: Linux 系统编程百科全书

## 核心内容
- 文件 I/O：open/read/write/close
- 文件系统：inode、权限、目录操作
- 进程管理：fork/exec/wait、进程生命周期
- 内存分配：brk/mmap、栈/堆
- 信号：signal handlers、sigaction、实时信号
- 进程间通信：pipe/FIFO/msgqueue/semaphore/shared memory
- 线程：pthread、线程同步
- Socket 编程：TCP/UDP、Unix domain、raw socket
- 时间：timer、clock、sleep
- 高级 I/O：epoll/select/poll、nonblocking I/O
- 资源限制与进程调度
- 虚拟文件系统
- 内存映射文件
