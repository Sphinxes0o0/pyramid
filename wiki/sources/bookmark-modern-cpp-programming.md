# Modern C++ Programming

## Overview

Federico Busato 的 C++ 课程，从基础到高级，29 章节，被 Verona 大学学术采用和 NVIDIA 专业培训使用。独特价值在于**专业软件工程技能**——这些在实际课程中很少覆盖。

> "Covers the basics of C++ programming and moves on to advanced C++ semantics and concepts"

## Core Content

### 29 Chapters

**基础 (Foundations):**
- Type system, operators, memory concepts
- Control flow, functions, preprocessing

**OOP:**
- Class hierarchy, polymorphism, operator overloading

**模板与元编程:**
- Function/class templates
- SFINAE
- C++20 Concepts

**翻译单元:**
- Linkage, ODR, Modules, Libraries

**专业开发 (Professional Development):**
- Code conventions
- Debugging
- Testing
- Ecosystem tools

**标准库:**
- Containers, iterators, algorithms
- Ranges
- Utilities

**高级语义:**
- Move semantics
- Smart pointers
- Concurrency

**性能 (Performance):**
- CPU architecture
- Optimization techniques
- Benchmarking
- Profiling

**软件设计:**
- SOLID principles
- Design patterns
- Idioms

**构建工程 (Build Engineering):**
- Binary size optimization
- Build time reduction

## 独特价值

### 专业工程技能（一般 C++ 教程缺失）

| 话题 | 价值 |

| **二进制大小优化** | 嵌入式/移动端关键技能 |
| **构建时间优化** | 大型项目效率 |
| **Benchmarking/Profiling** | 性能调优方法论 |
