# Linux高性能服务器编程

## 核心内容

中文 Linux 服务器编程书籍，主要涵盖：

- **TCP/IP 协议栈**深入剖析
- **Socket 编程** API 与最佳实践
- **I/O 模型**：select/poll/epoll/kqueue 等
- **并发模型**：多进程、多线程、异步 I/O
- **服务器架构**：常见模式与性能优化
- **网络协议**：HTTP/DNS/FTP 等应用层协议实现

## 关键要点

- 侧重 x86-64 Linux 环境
- 理论与实践结合，含代码示例
- 适合作为网络编程进阶读物
