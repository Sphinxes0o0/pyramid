# 编程面试书籍合集

## 书目一览

### 1. Cracking the Coding Interview, 6th Edition
- **作者**: Gayle Laakmann McDowell
- **出版**: CareerCup, 2015
- **版本**: 6th Edition
- **页数**: 708
- **内容**: 189道编程面试题及详解，涵盖数组/字符串、链表、树/图、递归/DP、排序/搜索、系统设计等全部面试主题
- **章节结构**:
  - 面试流程与策略（简历、行为问题、技术问题）
  - 数据结构专题（数组/字符串、链表、栈/队列、树/图）
  - 算法专题（递归、排序、搜索、DP）
  - 系统设计与可扩展性
  - 进阶主题（OOD、数据库、线程与锁）
  - 189道精选面试题
- **特点**: 每道题都有Hint + Solution + 复杂度分析；大厂面试真题来源
- **状态**: PDF，完整文本

### 2. Elements of Programming Interviews (EPI) — Sampler
- **作者**: Adnan Aziz (UT Austin/Google), Tsung-Hsien Lee (Google), Amit Prakash (ThoughtSpot/Google/Microsoft)
- **出版**: 2014, Version 1.4.4
- **页数**: 191 (sampler)
- **内容**: EPI面试指南概览，涵盖21章问题解决模式、数据结构与算法题、领域特定问题、荣誉题
- **Part I: 数据结构与算法**（16章）
  - Primitive Types、Arrays、Strings、Linked Lists、Stacks/Queues
  - Binary Trees、Heaps、Searching、Hash Tables、Sorting
  - BST、Recursion、DP、Greedy、Graphs、Parallel Computing
- **Part II: 领域问题** — 设计问题、语言问题、OOD、常用工具
- **Part III: 荣誉题** — 进阶问题
- **特点**: 结构化方法论（brute-force→optimized），所有代码Python 3
- **状态**: PDF完整sampler
