# 算法与数据结构书籍合集

## 书目一览

### 1. Algorithms in C, 3rd Ed — Part 5: Graph Algorithms
- **作者**: Robert Sedgewick (Princeton University)
- **出版**: Addison-Wesley, 2002
- **内容**: 图算法专题卷，涵盖DFS/BFS、有向图、最小生成树（Prim/Kruskal）、最短路径（Dijkstra/Floyd）、网络流（Ford-Fulkerson）等
- **语言**: C语言实现
- **状态**: 扫描版PDF，图算法全

### 2. Algorithms in C
- **作者**: Robert Sedgewick
- **出版**: Addison-Wesley, 3rd Ed
- **页数**: 672
- **内容**: 算法全卷（含排序、搜索、字符串、几何等），C语言实现
- **状态**: 图片扫描版，无直接文本

### 3. 数据结构（C++语言版）第3版 — dsacpp-3rd-edn
- **作者**: 邓俊辉（清华大学）
- **出版**: 清华大学出版社, 2013
- **ISBN**: 清华985名优教材
- **内容**: 数据结构C++实现，涵盖向量、列表、栈/队列、二叉树、图、排序、散列等
- **状态**: PDF文本部分可读

### 4. Data Structures & Algorithms in C++, 2nd Ed
- **作者**: Michael T. Goodrich, Roberto Tamassia, David Mount
- **出版**: 2nd Edition
- **内容**: 数据结构和算法的C++标准实现，涵盖基本数据结构到高级图算法
- **状态**: PDF，基础文本可用

### 5. The Art of Computer Programming, Vol.1: Fundamental Algorithms, 3rd Ed
- **作者**: Donald E. Knuth
- **出版**: 3rd Edition
- **内容**: 计算机程序设计艺术第一卷：基本算法（MMIX指令集、信息结构、数学预备知识）
- **地位**: 计算机科学史上最经典的算法著作
- **状态**: PDF

### 6. 我的第一本算法书
- **作者**: 石田保辉, 宫崎修一
- **内容**: 插图丰富的算法入门书，以图解方式讲解排序、搜索、图算法、安全算法等
- **语言**: 中文
- **状态**: PDF

### 7. 基础集合论
- **作者**: 中文教材
- **页数**: 244
- **内容**: 集合论基础：集合运算、关系、函数、基数等，为算法分析提供数学基础
- **状态**: 图片扫描版
