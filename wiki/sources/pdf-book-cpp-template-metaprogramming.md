# C++ Template Metaprogramming 中文版

## 核心内容

- **模板基础**：特化/偏特化、模板参数推导、SFINAE
- **类型系统**：type traits、conditional、enable_if、void_t
- **编译期计算**：constexpr 函数、整数序列、fold expression
- **惰性求值**：模板元编程的控制流（递归/迭代展开）
- **表达式模板**：重载运算符构建 AST
- **C++20 进展**：概念(Concepts)、requires 表达式、模板参数包展开
- **实践陷阱**：编译器限制、错误信息可读性、长编译时间
