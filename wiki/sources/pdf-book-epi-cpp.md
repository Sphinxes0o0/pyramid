# Elements of Programming Interviews

## 核心内容

- **300+ 题目**：精选算法/数据结构面试题，每题有最优解与多种变体
- **语言**：C++ 实现（也有 Java/Python 版），强调 STL 正确使用
- **章节结构**：数组、字符串、栈队列、树、图、动态规划、排序搜索等
- **题目按难度分级**：Easy / Medium / Hard，配详细图解与边界case分析
- **模板函数**：binary search、dfs、topological sort 等高频模板直接可抄
- **面试策略**：如何分析问题、如何 corner case、如何沟通思路

## 关键引用
