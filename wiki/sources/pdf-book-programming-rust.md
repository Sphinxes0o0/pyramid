# Programming Rust, 2nd Edition

## 核心内容

- **所有权系统**：Borrow checker、生命周期、Lifetime elision
- **类型系统**：Trait、泛型、Newtype pattern、PhantomData
- **错误处理**：Result/Option、? 运算符、自定义错误类型
- **并发**：Fearless Concurrency、Send/Sync、Arc/Mutex/RwLock
- **FFI 与 unsafe**：与 C 互调、Raw pointer、extern 块
- **标准库**：Iterator、Iterator trait、智能指针、Box/Rc/RefCell
- **生态**：Cargo/Clippy/cargo-fuzz、wasm-bindgen、async/await 入门
- **与 C++ 对比**：现代 Rust 写法 vs 等效 C++20

## 关键引用

> "Rust's ownership system is the key to memory safety without garbage collection, and concurrency without data races."
