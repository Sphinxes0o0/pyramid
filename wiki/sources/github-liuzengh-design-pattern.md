# Design Patterns In Modern C++ 中文版

GitHub: https://github.com/liuzengh/design-pattern

## 内容

23 种 GoF 设计模式的现代 C++ 实现，涵盖：

**创建型**: Builder, Factory, Prototype, Singleton
**结构型**: Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy
**行为型**: Chain of Responsibility, Command, Interpreter, Iterator, Mediator, Memento, Null Object, Observer, State, Strategy, Template Method, Visitor

## 技术栈

- C++11/14/17
- CMake 构建
- 每个模式独立 .cpp 文件
