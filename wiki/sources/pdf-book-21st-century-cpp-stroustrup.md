# 21st Century C++

## Core Content
Bjarne Stroustrup 关于现代 C++ 设计的 CACM 论文（26页）。

**五大核心理念**：
1. **Resource Management** — RAII，自动资源管理
2. **Lifetime Management** — 生命周期安全，防止 dangling pointer
3. **Error Handling** — 异常 vs 返回码的选择
4. **Modularity** — 模块化设计，接口与实现分离
5. **Generic Programming** — 泛型编程，类型安全的抽象

**确保代码现代化**：
- C++ Core Guidelines — 编程规范集
- Profiles — 类型安全、边界安全、资源安全的 profile 机制
- 从旧式 C++ 迁移到现代 C++ 的策略

**核心观点**：现代 C++ 比旧式 C++ 更简单、更安全、更快——不应再用 1985 年的方式写 C++。
