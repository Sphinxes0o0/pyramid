# Blood Flow Restriction Training (BFR, 血流限制训练)

## Definition

在肢体近端使用充气袖带限制静脉回流同时维持动脉血流进入的阻力训练方法，可在低负荷 (20-40% 1RM) 下诱导类似高强度训练的肌肉适应。

起源于日本 KAATSU，由 Dr. Yoshiaki Sato 于 1966 年发明。

## Mechanism

```
BFR + 低强度运动 →:
1. 代谢产物积累 (乳酸、H⁺、无机磷酸盐)
2. 生长激素 (GH) 增加
3. [[entities/exercise-science/physiology/mtor-pathway]] 激活
4. [[entities/exercise-science/physiology/satellite-cells]] 增殖
5. 肌肉蛋白合成增加
```

**生理效应**:
- **静脉阻断**: 限制血液回流，积累代谢物
- **动脉维持**: 允许动脉血流进入
- **代谢应激**: 缺氧环境促进生长激素分泌

## Scientific Evidence

**Scientific Reports Meta 分析 (2024)**: 28 项研究，542 名运动员 (14-26 岁)
- 力量: 效果量 0.74-1.03（小-大改善）
- 爆发力: 效果量 0.46
- 速度: 效果量 0.54
- 耐力: 效果量 1.39-1.40
- 体成分: 显著改善
- 体重: 无显著影响

## Parameters

| 类型 | 压力范围 |
|-----|---------|
| 袖带宽度 | 3-5 cm |
| 上肢压力 | 80-240 mmHg |
| 下肢压力 | 100-280 mmHg |
| 目标 | 动脉闭塞压 40-80% |

**训练方案**:
- 每周频率: 2-4 次
- 每组次数: 15-30 次
- 组数: 3-4 组
- 休息间隔: 30-60 秒

## Use Cases

**适用人群**:
- 康复患者
- 老年人
- 关节损伤恢复期
- 无法进行高强度训练者

**优势**:
- 仅需 20-40% 1RM 即可产生合成代谢效果
- 降低关节压力，减少损伤风险

**注意事项**:
- 心血管疾病患者慎用
- 需专业指导
- 避免过度加压
