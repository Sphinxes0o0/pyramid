# Muscle Hypertrophy (肌肥大)

## Definition

肌肉在阻力训练刺激下发生的肌纤维体积增大，是骨骼肌对机械张力和代谢应激的适应结果。

## Key Points

- **三大机制** (Schoenfeld 模型):
  - **机械张力 (Mechanical Tension)**: 主要驱动力，通过 [[entities/exercise-science/physiology/mtor-pathway]] 激活蛋白质合成
  - **代谢应激 (Metabolic Stress)**: 乳酸、H⁺、无机磷酸盐积累 → 细胞肿胀 + MAPK/ERK 通路激活
  - **肌肉损伤 (Muscle Damage)**: 肌纤维结构性损伤 → [[entities/exercise-science/physiology/satellite-cells]] 激活 → 修复-超量恢复
- **最优负荷范围**: 60-85% 1RM，每组 8-12 次，3-6 组
- **训练后 MPS 升高持续**: 24-72 小时
- **合成代谢抵抗**: 老年人对合成代谢刺激响应减弱，需要更高蛋白质剂量或亮氨酸富集
- **区域肌肥大**: 不同肌纤维区域对训练刺激响应存在差异

## Molecular Signaling

```
机械张力 → PI3K → Akt → [[entities/exercise-science/physiology/mtor-pathway]] → 蛋白质合成
                                         ↓
                                   4E-BP1 磷酸化
                                         ↓
                                   eIF4E 释放 → 翻译启动
```

## Nutrition Support

- 蛋白质摄入: 1.6-2.2 g/kg/天
- 亮氨酸: 每餐 2-3g（[[entities/exercise-science/physiology/mps-muscle-protein-synthesis]] 阈值 ~2-3g/餐）
- 训练后蛋白质时机: 0-2 小时窗口期
