# Concurrent Training (并发训练)

## Definition

同时进行有氧训练和阻力训练的训练方法，传统观点认为两者会相互干扰（协调效应），但 2024-2026 年研究表明干扰效应被夸大。

## The Interference Effect

### Traditional View
同时进行有氧和阻力训练会干扰彼此的适应，即"协调效应"。

### Updated Evidence (2016+)
- Sports Medicine 系统综述: 30+ 年的观念被重新审视
- 证据表明干扰效应被夸大
- 特定条件下并发训练可能促进肥大

## Molecular Mechanism: AMPK vs mTOR

```
有氧训练 → AMPK 激活 → 抑制 mTORC1 → 蛋白质合成↓
阻力训练 → mTORC1 激活 → 蛋白质合成↑
```

**问题核心**: 有氧训练可能干扰 [[entities/exercise-science/physiology/mtor-pathway]] 通路

**实际影响**: 取决于训练顺序、强度和类型

## Optimizing Concurrent Training

### Training Order

| 方案 | 安排 |
|-----|------|
| A (推荐) | 阻力 → 有氧 |
| B | 有氧 → 阻力 |
| C | 分开不同天进行 |

### Aerobic Type Selection

- 低强度有氧 (LISS): 干扰较小
- HIIT: 需谨慎安排
- 自行车 vs 跑步: 对腿部肌肉影响不同

### Frequency

- 每周 2-3 次有氧
- 阻力训练 3-4 次
- 留出足够恢复时间

## Population-Specific Findings

| 人群 | 发现 |
|-----|------|
| 乳腺癌幸存者 (2024) | 并发训练显著改善心肺功能，减少癌症相关疲劳 |
| 高血压患者 | 降低血压变异性，男女反应不同 |
| 儿童 1 型糖尿病 | 改善炎症因子，调节血糖相关激素 |

## Key Takeaway

协调效应可能不如传统观点认为的那么显著。通过合理安排（顺序、强度、频率），可以最大化两种训练的效果。
