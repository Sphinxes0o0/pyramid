# 运动肌贴 (Kinesiology Tape)

## 定义

运动肌贴（Kinesiology Tape, KT）是一种100%纯棉纤维的弹性医用胶带，可拉伸至原始长度的120-140%，模仿皮肤弹性设计，通过皮肤机械感受器刺激发挥神经调节作用。

## 关键要点

- **材料特性**: 弹性120-140%、厚度约0.5mm、低过敏性丙烯酸酯黏合剂、可持续佩戴24-72小时
- **作用机制**: 皮肤感受器刺激 → Gate Control Theory疼痛调节 → 可能的本体感觉增强
- **临床证据**: 对慢性踝关节不稳定有中等稳定效应(Level 1a)；对膝关节骨关节炎疼痛有效(Level 1a)
- **局限性**: 即时肌力提升效果不一致；安慰剂效应可能主导部分主观改善

## 作用机制

**神经生理学途径**:
```
肌贴拉伸 → 皮肤机械感受器激活(鲁菲尼小体、帕西尼小体)
    ↓
本体感觉输入增加
    ↓
脊髓和皮质水平感觉整合改善
    ↓
运动单位募集效率可能提高
```

**Gate Control Theory**: Aβ纤维激活可抑制C纤维痛觉传导

**H-reflex研究**(Bagheri 2018): 肌贴增强H-reflex参数（H斜率↑、H阈值↓、最大H反射强度↑），但表面EMG研究未发现肌肉激活差异

## 临床应用证据

### 有效应用场景

| 适应症 | 效应 | 证据等级 | 推荐程度 |
|--------|------|----------|----------|
| 慢性踝关节不稳定 | Y平衡↑、步态改善 | Level 1a | ★★★★★ |
| 膝关节骨关节炎 | 疼痛↓(SMD=-0.42)、等速肌力↑(SMD=0.72) | Level 1a | ★★★★☆ |
| 髌腱病 | 疼痛↓ | Level 1b | ★★★☆☆ |
| 老年人跌倒预防 | 平衡/本体感觉↑ | Level 2b | ★★★☆☆ |
| 肩袖康复(辅助) | 短期疼痛↓ | Level 1b | ★★☆☆☆ |

### 效果存疑/不推荐场景

| 场景 | 结论 |
|------|------|
| 即时肌力提升 | 多数高质量研究显示无改善 |
| 竞技运动表现 | 部分研究显示反而降低跳跃表现 |
| 肩袖康复 | 添加到运动方案无额外获益 |
| 术后疼痛管理 | 未显示减少阿片类药物使用 |

## 贴扎技术要点

- **张力**: 15-40%拉伸（研究显示此范围有效）
- **方向**: 证据不支持传统"从起点到止点"理论
- **持续时间**: 48小时或单次使用均有研究支持
- **弹性方向**: 足底贴扎建议选择无弹性或侧向弹性方向

## 安慰剂效应

2015年欺骗性对照试验显示：真肌贴、伪肌贴、无肌贴三组在峰值扭矩、总做功上**无显著差异**，提示安慰剂效应可能是即时改善的主要原因。
