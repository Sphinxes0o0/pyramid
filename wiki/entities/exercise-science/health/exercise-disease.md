# 运动与疾病预防 (Exercise & Disease Prevention)

## 定义

规律运动是心血管疾病、2型糖尿病、代谢综合征等慢性病一级和二级预防的有效手段。

## 糖尿病与运动

### 1型糖尿病

- 运动前血糖管理
- 碳水化合物/胰岛素剂量调整
- 专业医疗指导

**研究** (Pediatric Diabetes 2023): 并发训练改善1型糖尿病儿童炎症因子和血糖相关激素调节

### 2型糖尿病

**运动处方**:

| 参数 | 推荐 |
|------|------|
| 有氧强度 | 50-70% VO2max |
| 有氧时间 | 150-300分钟/周 |
| 阻力训练 | 2-3次/周，60-80% 1RM |
| 频率 | 每天或几乎每天 |

**机制**:
- GLUT4转运蛋白上调 → 胰岛素敏感性改善
- 肌肉糖原合成增加
- 肝脏胰岛素敏感性改善
- 运动降低TNF-α、IL-6系统性炎症

## 心血管健康与运动

### ACSM 指南 (2024)

- 定期身体活动对心血管健康必要
- 活动量与心血管益处呈剂量效应
- 即使少量活动也有益处

### 运动对心血管的益处

**结构改善**:
- 心肌质量增加
- 心脏容积扩大
- 毛细血管密度增加

**功能改善**:
- 静息心率降低
- 心输出量增加
- 血管内皮功能改善

**代谢改善**:
- 血脂改善 (LDL↓, HDL↑)
- 血压降低
- 胰岛素敏感性改善

### 高血压与运动

并发训练（有氧+阻力）可有效降低血压，男女反应可能不同，需长期坚持。

### 动脉僵硬度

- 运动降低动脉僵硬度
- 有氧和阻力训练均有效

## 代谢综合征

运动是代谢综合征管理的核心干预手段：

| 指标 | 运动效果 |
|------|---------|
| 血糖 | -12% |
| 胰岛素 | -50% |
| 胰岛素敏感性 | +55% |
| 体重 | -10% |
| 脂肪 | -14% |
| 甘油三酯 | -51% |
| HDL-C | +13% |

## 安全注意事项

**运动前评估**:
- 久坐不动者逐渐增加活动
- 高危人群需医学评估

**血糖监测（糖尿病患者）**:
- 运动前检测血糖
- 运动中必要时监测
- 运动后监测调整

**预防低血糖**:
- 餐后运动
- 随身携带碳水化合物
