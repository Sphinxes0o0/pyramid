# 跑步经济性 (Running Economy)

## 定义

跑步经济性（Running Economy, RE）是耐力表现的三大决定因素之一，定义为以给定速度亚极限跑步时的能量需求，通常表达为每跑1公里身体所需的摄氧量 (mL/kg/km)。

**RE越低越好** — 表示以给定速度跑步时氧气消耗更少

## 三大耐力关键生理因素

1. **VO2max** — 最大摄氧量
2. **LT** — 乳酸阈值
3. **RE** — 跑步经济性（本文主题）

## 影响因素

| 因素类别 | 具体因素 |
|---------|---------|
| 代谢因素 | 线粒体密度、酶活性 |
| 心肺因素 | 肺扩散能力、VO2 kinetics |
| 生物力学因素 | 步频、步幅、跑步技术 |
| 神经肌肉因素 | 肌纤维类型、神经驱动 |

## 改善方法

### 力量训练

*Sports Medicine* (2024) Meta分析：

| 训练类型 | 对RE的影响 |
|---------|-----------|
| 高负荷力量训练 | 正向改善 |
| 增强式训练 (Plyometric) | 正向改善 |
| 组合训练 | 协同效应 |

### 增强式训练

**Scientific Reports** (2023) 随机对照试验:
- 30名业余跑者，6周每日跳跃训练（双腿跳跃，5分钟/天）
- 显著改善跑步经济性

### 高强度间歇训练

*Sports Medicine* (2014) 综述: 上坡跑和场地高强度间歇训练可通过提高VO2 kinetics改善RE

### 高原训练

- 空气稀薄刺激红细胞生成，提高运氧能力
- 可改善RE达3-5%

### 下坡跑训练

- 增强腿部力量
- 提高步频效率
- 改善落地经济性

## 步态优化参数

| 参数 | 理想范围 |
|-----|---------|
| 步频 | 170-185步/分钟 |
| 触地时间 | 越短越好 |
| 腾空时间 | 与速度相关 |
| 步幅 | 适度，避免过大（overstriding）|

## 跑步姿势关键要素

- 身体前倾角度
- 手臂摆动
- 核心稳定性
- 腿部回收

## 测试方法

**实验室**: 亚极限VO2测试（固定速度下的VO2测量）、递增负荷测试
**现场**: 固定速度下的心率监测、运动手表数据
