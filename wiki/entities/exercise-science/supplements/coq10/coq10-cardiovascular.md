# CoQ10 & Cardiovascular System (辅酶Q10与心血管系统)

## Definition

辅酶Q10通过改善心肌能量代谢、抗氧化和抗炎作用，对心力衰竭、高血压和冠心病发挥心血管保护作用，是高耗能心脏组织的关键能量支持。

## Heart Failure — "Cardiac Energy Crisis"

```
心肌能量代谢障碍：
    ↓
ATP产生减少 → 磷酸肌酸↓ → 心肌收缩力↓
    ↓
线粒体功能障碍 → 电子传递链效率↓
    ↓
ROS产生↑ → 氧化损伤
    ↓
CoQ10水平↓
    ↓
补充CoQ10 → 纠正缺乏 → 心肌能量状态改善
```

## CoQ10 in Heart Failure

| 结局 | CoQ10效应 |
|------|----------|
| 左室射血分数（LVEF）| 显著增加 |
| 6分钟步行距离 | 增加 |
| NYHA心功能分级 | 改善 |
| 全因死亡率 | 降低（Q-SYMBIO试验）|

### Q-SYMBIO Trial Landmark Findings

- **设计**: 420例慢性心力衰竭患者，CoQ10 100mg 3次/日 vs 安慰剂，2年
- **发现**: 全因死亡率降低；心血管死亡率降低；因心衰住院率降低

## Hypertension

| 机制 | CoQ10作用 |
|------|----------|
| 血管舒张 | 激活K⁺通道，血管平滑肌松弛 |
| 抗氧化 | 减少血管ROS，改善内皮功能 |
| 抗炎 | 抑制NF-κB，血管炎症↓ |
