# Curcumin & Inflammation (姜黄素与炎症)

## Definition

姜黄素通过抑制NF-κB和MAPK通路、下调促炎细胞因子，对类风湿性关节炎（RA）等自身免疫性疾病发挥抗炎作用。透明质酸（HA）修饰纳米粒可实现主动靶向炎症关节。

## Rheumatoid Arthritis (RA) Pathology

```
自身抗原（瓜氨酸蛋白）
       ↓
  免疫耐受破坏
       ↓
  自身抗体产生（RF、抗CCP抗体）
       ↓
  免疫细胞浸润关节滑膜
       ↓
  促炎细胞因子释放（TNF-α、IL-1β、IL-6）
       ↓
  关节滑膜增生、血管翳形成
       ↓
  软骨和骨破坏（骨侵蚀）
```

## Key Targets

| 靶点/通路 | 姜黄素作用 | 结果 |
|---------|----------|------|
| **NF-κB** | 抑制激活 | TNF-α↓、IL-1β↓、IL-6↓ |
| **MAPK** | 抑制磷酸化 | MMP2/MMP9↓ → 软骨降解↓ |
| **RANKL/RANK** | 抑制通路 | 破骨细胞分化↓ → 骨侵蚀↓ |
| **Nrf2** | 激活 | 抗氧化酶↑ → ROS↓ |

## Nanoparticle Targeting Strategy

| 问题 | 解决方案 |
|------|---------|
| 口服吸收差 | 纳米乳剂/泡囊 |
| 无靶向性 | 透明质酸（HA）修饰 → 主动靶向CD44（炎症关节高表达）|
| 快速代谢 | 纳米载体保护 |

**Hyalo-Nio-CUR系统**：
- HA外壳 → CD44受体配体 → 主动靶向炎症关节
- 泡囊双分子层 → 包封姜黄素 → 提高稳定性
- 临床前数据：MDA↓、IL-1β↓、IL-6↓、MMP2↓、RANKL↓

## Curcumin Anti-RA Mechanism

```
透明质酸纳米粒递送姜黄素 → 姜黄素靶向富集于关节
       ↓
NF-κB抑制 → TNF-α↓、IL-1β↓、IL-6↓
       ↓
COX-2抑制 → PGE₂↓
       ↓
关节炎症↓、肿胀↓
```
