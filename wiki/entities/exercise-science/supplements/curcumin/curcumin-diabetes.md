# Curcumin & Diabetes (姜黄素与糖尿病)

## Definition

姜黄素通过多靶点作用改善2型糖尿病（T2DM）患者的胰岛素敏感性、β细胞功能和血糖控制。

## Key Targets

| 靶点 | 作用 | 结果 |
|------|------|------|
| **GSK-3β** | 抑制活性 | 恢复糖原合酶活性 → 葡萄糖利用↑ → 血糖↓ |
| **IAPP** | 抑制聚集 | 减少β细胞淀粉样沉积 → β细胞保护 → 胰岛素分泌↑ |
| **β细胞** | 抗氧化/抗炎保护 | β细胞凋亡↓ → β细胞量/功能↑ |
| **AMPK** | 激活 | 改善β细胞能量代谢；脂肪氧化↑ |
| **PPAR-γ** | 激活 | 胰岛素敏感性↑ |

## Mechanism Map

```
姜黄素（多靶点）
     │
     ├─→ 抑制GSK-3β
     │       ↓
     │   激活糖原合酶
     │       ↓
     │   糖原合成↑ → 血糖利用↑
     │
     ├─→ 抑制IAPP聚集/毒性
     │       ↓
     │   β细胞保护
     │       ↓
     │   胰岛素分泌↑
     │
     ├─→ 抗氧化 → β细胞氧化应激↓
     │
     ├─→ 抗炎 → β细胞炎症损伤↓
     │
     └─→ 激活AMPK → 改善能量代谢
             │
             ↓
     胰岛素抵抗改善 → T2DM风险↓
```

## Clinical Evidence

| 研究 | 设计 | 主要发现 |
|------|------|---------|
| Nutrition Journal 2024 | RCT, n=272, 12个月, 1500mg/天 | 显著改善β细胞功能（HOMA-β↑）；HbA1c↓；空腹血糖↓ |
| Nutrients 2020 | RCT, T2DM高风险成人 | 循环GSK-3β和IAPP水平显著降低 |

### Nutrition Journal 2024 — 关键数据

- **样本**: 272名肥胖T2DM患者
- **干预**: 姜黄素1500mg/天 vs 安慰剂，12个月
- **设计**: 随机、双盲、安慰剂对照
- **结果**: β细胞功能显著改善；空腹血糖降低
