# Curcumin & Liver / MAFLD (姜黄素与肝脏/代谢相关脂肪肝病)

## Definition

姜黄素通过抗氧化、抗炎、抗脂肪肝和抗纤维化多途径改善代谢相关脂肪性肝病（MAFLD，原NAFLD/NASH），延缓疾病进展。

## Disease Spectrum

```
MAFLD进展
脂肪变性 → MASH(NASH) → 纤维化 → 肝硬化 → HCC
```

## Key Targets

| 靶点/通路 | 姜黄素作用 | 结果 |
|---------|----------|------|
| **SREBP1c↓** | 抑制脂肪酸合成 | 减少肝细胞脂肪积累 |
| **PPAR-α↑** | 促进脂肪酸β氧化 | 脂肪代谢↑ |
| **Nrf2激活** | 上调SOD、GPx、CAT、HO-1 | ROS↓ → 氧化损伤↓ |
| **NF-κB抑制** | 下调TNF-α、IL-6、IL-1β | 肝脏炎症↓ |
| **HSC活化抑制** | 阻断TGF-β/Smad | 胶原沉积↓ → 纤维化↓/延缓 |
| **AMPK激活** | 抑制mTOR | 脂肪合成↓；自噬↑ |

## Mechanism Map

```
姜黄素对MAFLD的保护作用
       │
       ├─【抗氧化】
       │   激活Nrf2通路 → 抗氧化酶↑（SOD、GPx、CAT）
       │
       ├─【抗炎】
       │   抑制NF-κB → TNF-α↓、IL-6↓、IL-1β↓
       │
       ├─【抗脂肪肝】
       │   SREBP1c↓（脂肪酸合成↓）
       │   PPAR-α↑（β-氧化↑）
       │
       └─【抗纤维化】
           抑制肝星状细胞（HSC）活化
           → 胶原沉积↓ → 纤维化↓
```

## Clinical Evidence

| 研究 | 设计 | 主要发现 |
|------|------|---------|
| Scientific Reports 2023 | 荟萃分析，16个RCT + 6个SRMA | ALT降低5.61 IU/L（MD=-5.61, 95%CI: -9.37,-1.85）；AST降低3.90 IU/L（MD=-3.90, 95%CI: -5.97,-1.82）|
| Archives Medical Science 2018 | 系统综述 | 明确的抗纤维化机制证据 |

### Clinical Meaning of ALT/AST Reduction

| 指标 | 正常值 | 姜黄素组变化 | 意义 |
|------|--------|------------|------|
| ALT | 7-56 IU/L | **-5.61 IU/L** | 肝细胞损伤减轻 |
| AST | 10-40 IU/L | **-3.90 IU/L** | 肝细胞膜完整性改善 |
