# Curcumin & Kidney / Diabetic Nephropathy (姜黄素与肾脏/糖尿病肾病)

## Definition

姜黄素通过抑制足细胞间充质转分化（EMT）、诱导自噬和调节PI3K/Akt/mTOR通路，对糖尿病肾病（DN）发挥肾保护作用，延缓蛋白尿和肾功能恶化。

## Diabetic Nephropathy (DN) Progression

```
高血糖
    ↓
肾小球高滤过
    ↓
足细胞损伤 → 蛋白尿
    ↓
EMT转分化（失去足细胞特性）
    ↓
自噬缺陷
    ↓
系膜基质扩张 / 肾小球基底膜增厚
    ↓
结节性糖尿病肾小球硬化
    ↓
终末期肾病（ESRD）
```

## Podocyte EMT in DN

| 标志物变化 | 意义 |
|-----------|------|
| Nephrin↓ | 裂孔隔膜蛋白丢失 → 滤过屏障破坏 |
| Podocin↓ | 足细胞骨架蛋白丢失 |
| α-SMA↑ | 间充质marker → 上皮特性丧失 |
| Vimentin↑ | 间充质侵袭能力获得 |

## Key Mechanisms

```
姜黄素肾保护机制

姜黄素
    │
    ├─→ 抑制足细胞EMT
    │       ↓
    │   Nephrin↑，Podocin↑，α-SMA↓
    │       ↓
    │   蛋白尿↓
    │
    ├─→ 诱导自噬（AMPK激活，mTOR抑制）
    │       ↓
    │   受损蛋白质/线粒体清除
    │       ↓
    │   足细胞存活↑
    │
    └─→ 抑制凋亡
            ↓
        Caspase-3↓，Bcl-2↑
            ↓
        足细胞存活↑
            ↓
        肾功能改善
```

## PI3K/Akt/mTOR Pathway

| 状态 | 效应 |
|------|------|
| Akt激活 | 足细胞存活、抑制凋亡 |
| mTOR激活 | 抑制自噬、促进EMT（有害）|
| 姜黄素效应 | 激活PI3K/Akt → mTORC1抑制 → 自噬激活 + EMT抑制 |
