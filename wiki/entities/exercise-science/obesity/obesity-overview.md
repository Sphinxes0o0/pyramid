# Obesity Overview — 内脏脂肪与多系统损伤概述

## Definition

内脏脂肪（Visceral Adiposity Tissue, VAT）是位于腹腔、包裹脏器的脂肪组织，不同于皮下脂肪。当过量积累时，VAT从一个保护性储能组织转变为一个促炎性、促代谢紊乱性的活跃内分泌器官。

## VAT vs Subcutaneous Fat

| 脂肪类型 | 位置 | 功能 | 危害 |
|---------|------|------|------|
| **皮下脂肪** | 皮肤下层 | 储能、绝缘、激素储存 | 主要影响外观和活动能力 |
| **内脏脂肪** | 腹腔，包裹脏器 | 支撑保护脏器、活跃分泌脂肪因子 | **直接损害内脏功能** |

## Pathological Mechanisms

当内脏脂肪过量积累时，其功能从保护性内分泌转变为促炎性：

```
内脏脂肪功能障碍
        │
        ├─ 抗脂解功能受损 → 游离脂肪酸（FFA）持续溢出
        ├─ 促炎脂肪因子分泌↑ → IL-6、TNF-α、IL-1β等系统性释放
        ├─ 保护性脂肪因子分泌↓ → 脂联素显著降低
        ├─ 胰岛素抵抗加剧 → 细胞对胰岛素敏感性下降
        └─ 氧化应激增加 → ROS累积，细胞损伤
```

### Core Common Pathway: Chronic Low-Grade Inflammation

贯穿肥胖相关所有器官损伤的核心机制是**慢性低度炎症（Chronic Low-Grade Inflammation）**：

```
内脏脂肪过量 → 脂肪组织应激 → 炎症小体（NLRP3等）激活
        │
        ├─→ 全身炎症水平升高
        ├─→ 血管内皮功能障碍
        ├─→ 代谢紊乱（胰岛素抵抗、糖耐量异常）
        └─→ 免疫细胞浸润（巨噬细胞M1型↑）
```

### Three Core Pathways Shared Across All Systems

```
                    内脏脂肪过量积累
                          │
            ┌─────────────┼─────────────┐
            ↓             ↓             ↓
       慢性低度炎症    代谢紊乱      脂肪因子失衡
            │             │             │
     IL-6/TNF-α↑    胰岛素抵抗    脂联素↓
     NLRP3激活        糖毒性        瘦素↑
     氧化应激         脂毒性        RBP4↑
            │             │             │
            └─────────────┼─────────────┘
                          │
                   多器官系统性损伤
```

## Multi-System Impact Summary

| 器官系统 | 炎症介导的损伤 | 关键数据 |
|---------|--------------|---------|
| 心脏 | 动脉粥样硬化、心肌纤维化、心律失常 | 肥胖者房颤风险+40-90%；60岁以下无传统风险因素HR=1.44 |
| 肝脏 | 脂肪变性→MASH→纤维化、肝硬化 | 全球MASLD患病率32.4%；MASLD患者首要死因是CVD |
| 大脑 | 神经炎症、血脑屏障破坏、突触损伤 | 体脂率每增9.2%，认知衰老1年 |
| 胰岛 | β细胞凋亡、胰岛素分泌障碍 | VAI每增1单位，T2DM风险+43% |
| 肾脏 | 肾小管间质炎症、肾小球硬化 | VAI每翻倍，CKD风险+12% |

## VAI: A Composite Metabolic Health Index

内脏脂肪指数（VAI）整合了腰围、BMI、甘油三酯和HDL：

- **VAI翻倍** → CKD风险增加12%，T2DM风险增加43%
- 即使BMI正常，高VAI仍意味着代谢风险升高

## Treatment Thresholds

| 减重幅度 | 效果 |
|---------|------|
| ≥5% | 肝脏脂肪变性改善、代谢指标改善 |
| ≥7% | MASH可能缓解 |
| ≥10% | 纤维化可能逆转；房颤负担显著降低 |
