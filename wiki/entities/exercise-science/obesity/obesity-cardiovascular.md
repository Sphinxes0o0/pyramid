# Obesity & Cardiovascular System — 肥胖与心血管系统

## Definition

肥胖是心血管疾病（CVD）的**独立风险因素**，通过血流动力学改变、代谢紊乱、炎症反应和脂肪因子失衡四条通路损害心脏和血管系统。其效应在校正传统风险因素（高血压、高血脂、吸烟等）后依然显著。

## Epidemiological Evidence

| 指标 | 数据 |
|------|------|
| 西方国家成人肥胖患病率 | ~25% |
| 肥胖者高血压风险 vs 正常体重 | 2-3倍 |
| 肥胖者房颤风险 vs 正常体重 | 增加40-90% |
| 房颤归因于肥胖的比例 | ~20% |
| 无传统风险因素下肥胖CVD风险（60岁以下）| HR=1.44 (1.24-1.67) |

## Key Findings by Sub-Domain

### 1. Obesity Paradox — Critical Evaluation

**现象**：部分观察性研究发现肥胖心衰患者预后较好。

**批判性结论**（Current Heart Failure Reports, 2024）：
- 肥胖悖论主要源于**队列选择偏倚和残余混杂**
- 临床试验证据**不支持**"刻意保持肥胖对心衰有益"
- 体重减轻（尤其减重手术）确实改善心衰患者预后
- **不应以"肥胖悖论"为由忽视肥胖管理**

### 2. Heart Failure — Three Pathways

```
肥胖
  │
  ├─ 血流动力学 → 血容量↑ → 前/后负荷↑ → 心室重构 → HFpEF
  ├─ 代谢性 → 脂毒性 + 线粒体功能障碍 → 心肌能量危机
  └─ 炎症性 → 系统性炎症 → 心肌炎症 → 纤维化
```

**HFpEF（射血分数保留心衰）**是肥胖相关心衰的主要表型。

### 3. Atrial Fibrillation

**Obesity Reviews (2024, IF~8)** — 肥胖领域最权威期刊：

| 发现 | 详情 |
|------|------|
| 减重≥10% | 房颤负担显著降低 |
| 机制 | 左心房扩张+脂肪浸润→心房纤维化→电生理稳定性下降 |
| 双向关系 | 肥胖→房颤→运动耐量↓→体重↑→房颤加重 |
| 管理 | 目标导向减重是基石；术前减重可改善消融成功率 |

**Postoperative AF**（IJ Obesity, 2025）：腰围（而非BMI）是心脏手术后POAF的独立风险因素；POAF发生率20-40%。

### 4. Arterial Stiffness

Scientific Reports (2021, n=3,758)：四种体脂参数与baPWV（动脉僵硬度）的比较：

| 指标 | 多因素分析 | 结论 |
|------|-----------|------|
| BMI | NS (p=0.175) | ❌ 不能预测动脉硬化 |
| WC（腰围）| NS | ❌ 校正混杂后不显著 |
| **WHR（腰臀比）** | **显著** | ✅ 最佳预测指标 |
| **VFA（内脏脂肪面积）** | **显著** | ✅ 直接反映内脏脂肪 |

WHR > 0.90（男性）/ > 0.85（女性）与显著更高的动脉僵硬度相关。

## Mechanism Summary

```
内脏肥胖 → 血流动力学改变 + 炎症 + 脂毒 → 动脉硬化 + 心肌重构 → 心衰/房颤/心梗/卒中
```

## Treatment

| 干预 | 心血管获益 |
|------|-----------|
| GLP-1激动剂（Semaglutide）| MACE显著降低 |
| GIP/GLP-1双靶点（Tirzepatide）| 心衰住院↓ |
| SGLT-2抑制剂 | 心衰预后改善（独立于减重）|
| 减重手术 | 心衰症状改善、左心室重构逆转 |
