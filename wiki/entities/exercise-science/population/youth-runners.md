# 青少年跑者 (Youth Runners)

## 定义

青少年跑者指18岁以下的跑步参与者，其身体发育尚未完成（骨骼生长板未闭合、心肺系统发育中、协调性持续发展），需要特别关注训练强度和训练量以避免对成长造成不利影响。

## 关键要点

- **发育特点**: 骨骼生长板未闭合 → 应力性骨折和生长板损伤风险；心肺系统发育中；协调性持续发展
- **训练原则**: 以游戏和兴趣为主；避免专业化训练；循序渐进；多样化运动
- **年龄分级**: <12岁趣味为主 → 12-14岁<30km/周 → 15-16岁<50km/周 → 17-18岁<70km/周
- **研究价值**: 体能与执行功能正相关；心肺功能与认知灵活性相关；肌肉力量与注意力相关

## 体能评估 (PREFIT)

| 测试 | 评估内容 |
|-----|---------|
| 20m shuttle run | 心肺功能 |
| Handgrip | 上肢力量 |
| Standing long jump | 下肢力量 |
| 4×10m shuttle run | 灵敏性 |
| sit-and-reach | 柔韧性 |

## 训练原则

### 基本原则
- 以游戏和兴趣为主
- 避免专业化训练
- 循序渐进
- 多样化运动

### 技术发展重点
- 基本跑步姿势
- 呼吸技术
- 热身和放松
- 拉伸方法

### 力量训练（适合内容）
- 自重训练
- 基础力量动作
- 核心稳定性
- 避免大重量

## 损伤预防

### 高发问题
| 问题 | 原因 |
|------|------|
| 生长板损伤 | 过度冲击 |
| 应力性骨折 | 跑量过大 |
| 肌腱问题 | 肌肉力量不足 |

### 预防策略
- 充足的休息和恢复
- 避免单一运动过度
- 全身发展
- 定期评估

## 运动对儿童的益处

**认知发展**: 改善执行功能；提高注意力；促进学业表现

**心理健康**: 减少焦虑和抑郁；提高自尊；改善社交技能

**身体健康**: 维持健康体重；骨骼健康；心血管健康

## 身体活动指南

| 年龄 | 建议 |
|------|------|
| 3-5岁 | 全天活动 |
| 6-17岁 | 60分钟/天中高强度 |
