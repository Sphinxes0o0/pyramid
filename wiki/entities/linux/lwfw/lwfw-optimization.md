# LWFW Optimization Summary

## 定义

LWFW 优化建议汇总，基于对 lwfw/lwct/lwfw_agent 各模块的深度分析，按**性能**、**安全性**、**稳定性**、**代码质量**四个维度整理。

## 性能问题

| 优先级 | 问题 | 影响 | 建议 |
|--------|------|------|------|
| P2 | 事件合并 O(n²) | 512事件需130k次比较 | 哈希表 O(n) 合并 |
| P2 | 深拷贝持锁时间过长 | 规则多时阻塞数百毫秒 | RCU 无锁切换 |
| P3 | L2 过滤默认关闭 | VLAN/MAC 功能不可用 | 默认启用 |

## 安全性问题

| 优先级 | 问题 | 影响 | 建议 |
|--------|------|------|------|
| **P0** | 位标志冲突 | IP前缀和端口范围同时设置时相互覆盖 | 重构标志位定义 |
| P1 | 静态解析器状态 | 多线程并发调用时状态互相覆盖 | 状态移入 parser_state 结构体 |
| P2 | sscanf 格式串注入 | 负数或>255 octet 导致越界写入 | 使用 inet_pton() |

## 稳定性问题

| 优先级 | 问题 | 影响 | 建议 |
|--------|------|------|------|
| **P0** | 热重载期间 Ingress 无防护 | 重载期间防火墙完全失效 | 双策略切换 |
| P1 | 位操作非原子 | 连接状态位更新可能丢失 | __atomic_fetch_or |
| P2 | timer_thread 退出无重启 | 日志轮转和重载永久失效 | 添加线程监控 |
| P2 | 深拷贝无错误恢复 | 策略损坏时无法回滚 | 两阶段拷贝 |

## 代码质量问题

| 问题 | 说明 | 建议 |
|------|------|------|
| 日志宏重复定义 | `LWFW_PRINTF_*` 和 `LWCT_PRINTF_*` 大量重复 | 提取到共享头文件 |
| 初始化失败无 return | `lwfw_init()` 失败后继续执行 | 添加 return |
| 魔法数字 | `LWFW_TREE_BUCKET_SIZE=8` 等应可配置 | 移入配置参数 |

## 优化优先级总排序

| 优先级 | 问题 | 影响 |
|--------|------|------|
| **P0** | 位标志冲突、热重载窗口期、初始化无 return | 功能正确性 |
| **P1** | 静态解析器状态、深拷贝无恢复 | 稳定性 |
| **P2** | 事件合并 O(n²)、位操作非原子 | 性能/稳定性 |
| **P3** | 日志宏重复、魔法数字 | 代码质量 |
