# 动态规划 (Dynamic Programming)

## 定义

动态规划是一种**运筹学方法**，在多轮决策过程中寻求最优解。与分治法的核心区别：子问题**可以不相互独立**。

## 关键要点

### 动态规划核心概念

- **多轮决策**：每轮决策是一个子问题，子问题间可能有依赖
- **状态 (State)**：描述多轮决策过程的变量，受决策动作影响
- **状态转移方程**：sk+1 = uk(sk)，最核心的公式

### DP 适用问题的三个特征

1. **最优子结构**：原问题的最优解包含的子问题的解也是最优的
2. **无后效性**：今天的动作改变不了历史
3. **有重叠子问题**：子问题不独立（区别于分治法的关键）

### 解题步骤

1. **分阶段**：将原问题划分成几个子问题（阶段）
2. **找状态**：选择合适的状体变量 Sk
3. **做决策**：确定决策变量 uk
4. **状态转移方程**：写出 sk+1 = uk(sk)
5. **定目标**：写出指标函数 Vk,n
6. **寻找终止条件**：确定起点和终点

### 经典案例：最短路径

从 A 到 G 的最短路径问题，通过动态规划从后往前递推：
- 利用最优子结构：A→...→F1→G 最优，则 A→...→F1 也最优
- 最终路径：A → B1 → C2 → D1 → E2 → F2 → G，最短距离 18

## 动态规划 vs 分治法

| 特征 | 分治法 | 动态规划 |
|------|--------|---------|
| 子问题独立性 | 必须相互独立 | 可以不独立 |
| 子问题重叠 | 无 | 有 |
| 最优子结构 | 需要 | 需要 |

## 注意事项

- DP 问题没有标准化解题方法，需因题而异
- 不是所有岗位都会考察 DP，但它是面试难点
- 实际工作中使用频率并不高
