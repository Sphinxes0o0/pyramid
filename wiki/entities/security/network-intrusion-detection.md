# Network Intrusion Detection

## 定义

Network Intrusion Detection（网络入侵检测）是通过监控网络流量来检测恶意活动、攻击行为和策略违反的安全技术。

## 关键概念

- **NIDS vs HIDS**: Network-based IDS vs Host-based IDS
- **签名检测**: 基于已知攻击模式的特征匹配
- **异常检测**: 基于流量行为异常的检测
- **Snort**: 经典的开源网络入侵检测系统
